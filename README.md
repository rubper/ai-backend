## Description

This backend project based on the [Nest](https://github.com/nestjs/nest) framework, to assert feasibility on the idea to use tensorflow in a backend microservice.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
